Sobre o Projeto

Este site está sendo desenvolvido como parte do projeto da disciplina de Introdução à Computação, com o objetivo de aplicar os conhecimentos adquiridos em sala de aula sobre desenvolvimento web e integração com APIs. A proposta é criar um gerador de imagens a partir de descrições textuais, utilizando tecnologias como HTML, CSS e JavaScript.

O HTML está sendo utilizado para estruturar o conteúdo da aplicação, enquanto o CSS é responsável pelo layout e estilização da mesma.

Como Funciona?

  • Escreva um promp detalhando a imagem que deseja criar, como: "Crie um pássaro azul sem asas e com o bico longo"

  • Aperte o botão de "Gerar"

  • Quando o programa tiver finalizado, a tela será atualizada com a imagem pedida.
